# Music

A responsive Vue client to consume music ranks

View live demo at: [music.fakefarm.com](http://music.fakefarm.com)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

npm run test:unit
